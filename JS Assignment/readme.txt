Fundamental JavaScript Question

1. Hello, World!:
Write a program to print "Hello, World!" in the console.

2. Simple Calculator (Addition):
Write a program to take two numbers as input and display their sum.

3. Odd or Even:
Write a program that checks whether a number is odd or even.

4. Maximum of Two Numbers:
Write a program to find the maximum of two numbers provided by the user.

5. Multiplication Table:
Write a program to print the multiplication table of the number entered by the user.

6. Array Sum:
Write a program to calculate the sum of all elements in an array.

7. Count Vowels:
Write a function to count the number of vowels in a given string.

8. Reverse a String:
Write a program to reverse a string input by the user.

9. Factorial of a Number:
Write a function that takes a number and returns its factorial.

10. FizzBuzz:
Write a program that prints numbers from 1 to 50, but:
Print "Fizz" for multiples of 3,
Print "Buzz" for multiples of 5,
Print "FizzBuzz" for multiples of both 3 and 5.

Intermediate JavaScript Questions

1. Palindrome Checker:
Write a function to check if a string is a palindrome (reads the same forward and backward).

2. Unique Values in Array:
Write a program to remove duplicate values from an array and return the unique values.

3. Array Sorting:
Write a program to sort an array of numbers in ascending order without using built-in functions.

4. Longest word in a String:
Write a function that takes a sentence and returns the longest word in it.

5. Array Average:
Write a program to find the average of a ll numbers in an array.

6. Count Occurrences:
Write a program to count the occurrences of each character in a string.

7. Prime Number Checker:
Write a function to check if a number is prime.

8. Flatten Nested Array:
Write a function to flatten a nested array (e.g., [1, [2, [3, 4]], 5] should become [1, 2, 3, 4, 5]).

9. Anagram Checker:
Write a function to check if two strings are anagram of each other.

10. Frequency of Elements:
Write a program that takes an array and returns the frequency of each element in it.
